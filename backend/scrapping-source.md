### Sources à scrapper et API recommandées

Puisque vous êtes en **M1 SSI**, vous devez cibler des sources à haute valeur ajoutée technique.

| Source | Type | Intérêt |
| --- | --- | --- |
| **Reddit (r/netsec)** | API / RSS | Actualités techniques pures et vulnérabilités. |
| **Hacker News (YCombinator)** | API Algolia | Trends tech et sécurité globale. |
| **BleepingComputer** | RSS / Scraping | Alertes sur les malwares et ransomwares. |
| **The Hacker News** | Scraping | Actualités cyber-sécurité généralistes. |
| **NIST (NVD)** | API | Base de données des CVE (vulnérabilités critiques). |
| **Zataz** | Scraping | Veille sur les fuites de données (DataBreach). |

> [!TIP]
> **Outil recommandé :** Pour le scraping en Java, on utilise **Jsoup** pour les sites statiques. Si un site utilise beaucoup de JavaScript, tournez-vous vers **Playwright Java**.
